# YouTube Player
A YouTube Player with numerous features designed to provide a simple-to-use alternative to YouTube playlists that offers no interruptions and custom playlists all stored on your device.

## Requirements
1. The website must be the active tab in your browser window. If you wish to use another tab in the browser at the same time, move the player to a separate window.

2. It is strongly recommended that you use an ad blocker, such as uBlock Origin for Firefox and Edge or uBlock Origin Lite for Chrome, as YouTube loves to put ads in embeds.

## Notes on testing

This project is not tested very thoroughly on mobile devices or devices with small screens. Optimal performance may not be achievable on these devices.

Testing has mainly occurred on Firefox.

## How to use
Simply open the page and follow the instructions.

## Live version
A hosted version of the project is available on the website (https://iwltechnology.github.io/ytplayer).

## Features
- No interruptions, given that you use an ad blocker.
- Ability to save, rename and delete custom playlists.
- Ability to copy default playlists.
- Import videos from YouTube video link or playlist link.
- Ability to set up hot links to control multiple aspects of the website (see Menu -> URL Configuration Help on the website).
- Multiple themes
- Video data library (title, author) is cached and stored to local storage.
- Import / export playlists and the video data library.
