    URL Help

    URL options can be placed after a # in the address bar, and separated by a semi-colon.
    E.g [PAGE URL]#ytpl_theme:dark;ytpl_looping:off

    Current options are:
    ytpl_theme:[DARK OR LIGHT]. Force set the theme.
    ytpl_looping:[ON OR OFF]. Force set the looping setting.
    ytpl_playlist:[NUMBER or CNUMBER]. Enter the number of a playlist to automatically load. E.g 0 (the first button), or c0 (the first custom playlist).
    ytpl_shuffle. Shuffle the playlist. Must be placed after ytpl_playlist.
    ytpl_play_now. Play the chosen playlist with all other settings immediately on page load. Must be placed after ytpl_playlist and any other values.

    So, an example using all the above values (spaces added for clarity, don't use them) is [PAGE URL]#ytpl_theme:dark; ytpl_looping:off; ytpl_playlist:1; ytpl_shuffle; ytpl_play_now
